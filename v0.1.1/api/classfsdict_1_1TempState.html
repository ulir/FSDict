Detailed Description

template<typename TransTable_t__>
class fsdict::TempState< TransTable_t__ >

TempState is to be used to construct a state of the automaton bit by bit.

Once it is certain that no more outgoing edges need to be added, it can be stored into the Automaton. Edges can point to already-stored edges only.
