Detailed Description

template<MSMatchMode Mode = STANDARD>
class fsdict::MSMatch< Mode >

MSMatch provides approximate matching in large dictionaries using determinstic Levenshtein automata.

It is an implementation of the concepts described in "S. Mihov and K. Schulz. Fast approximate search in large dictionaries. Computational Linguistics, 30, 2004." Basically the implementation is derived from a C-implementation of Stoyan Mihov.

It uses the universal levenshtein automaton that is implemented in class LevDEA.

MSMatch implements the interface LevFilter, that is, it provides a method query taking a word as input and a CandidateReceiver as output argument. To use MSMatch, just implement a CandidateReceiver for your needs.

See Also
fsdict::LevDEA, fsdict::LevFilter, fsdict::LevFilter::CandidateReceiver
