Detailed Description

This is a custom implementation of a codecvt facet for utf-8, as it does not seem to be present on all systems (especially WIN).

All methods implemented here are implementations of virtual methods of std::codecvt. Please consult some STL documentation for info on the methods.

Author
Ulrich Reffle, 2008
