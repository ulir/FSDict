Detailed Description

template<typename TransTableType>
class fsdict::StateHash< TransTableType >

Statehash is a hashtable to store states of a MinDic during dictionary construction.

This allows for an efficient check if a new state is equivalent to an existing one.

Inserts a state into the hashtable.

This method does not check for duplicates!

This method does not check if an equivalent state is already in the hashtable - this must be done before, using StateHash::findState().

template<typename TransTableType>
template<typename TempState >
size_t fsdict::StateHash< TransTableType >::findState ( const TempState &  state)
inline

This method checks for a given TempState, if an equivalent state in the TransTable has been added to the hashtable before.

If so, the respective state-id in the transtable is returned. Otherwise, 0 is returned.

Returns
If a state equivalent to the given one exists, the respective transtable-state-id is returned. Otherwise, 0 is returned.

The comparison of a TempState and a state in the transTable is delegated to class TransTable, because the comparison depends on the TransTable type.
