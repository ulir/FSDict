Detailed Description

A specialised variant of the Aho-Corasick trie to search in the set of left or right sides of a PatternSet.

For the sake of simplicity and efficiency, the implementation details of class PatternSet are used here: Internally we use simple size_t variables to refer to a Pattern instead of using the official and public PatternRef interface.

Author
Ulrich Reffle, 2008
