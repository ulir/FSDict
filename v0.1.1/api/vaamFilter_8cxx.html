Detailed Description

Vaam.

vaamFilter is a command-line tool for the usage of the class Vaam. It is invoked with a distance bound k, a compiled minimized dictionary dic and a file containing a set of patterns P.

Please consult the documentation of class fsdict::Vaam for details.
